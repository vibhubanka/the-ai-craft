# Agent Config Examples

This folder contains example agent configuration files that can be used with various AI coding assistants (opencode, Claude Code, etc.). These configs help standardize tool behavior and workflows across your projects.

## Included Files

- [AGENTS.md](./AGENTS.md) — Agent instructions that enforce Conventional Commits and `gh` CLI usage for all GitHub workflows. Ideal for projects wanting standardized commit messages and a consistent GitHub CLI workflow.

## Usage

Copy the desired file to your agent's config directory:

- **opencode:** `~/.config/opencode/AGENTS.md`
- **Claude Code:** `~/.claude/CLAUDE.md` (project-local) or a global equivalent

Refer to your agent's documentation for more details on configuration.
