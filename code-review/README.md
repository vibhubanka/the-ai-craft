# Code Review Prompts

This folder contains carefully crafted prompts for AI-assisted code review. Each prompt targets a specific review scope and depth level.

## Included Prompts

- [7-Engineer Audit Review](./7-engineer-audit-review.prompt.md) — A comprehensive, multi-perspective production readiness audit prompt that covers security, architecture, frontend, DevOps, reliability, database, and AI safety. Ideal for pre-launch reviews and critical system evaluations.

## Usage

Pass the prompt file to your AI tool of choice along with your codebase. Follow the output format specified in each prompt for structured, actionable findings.
